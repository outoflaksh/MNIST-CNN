# MNIST-CNN

**Implementation of the deep learning algorithm Convultional Neural Network (or CNN) to classify images of handwritten digit samples in 10 classes (0-9).**

## Technologies used:
1. Keras
2. Tensorflow
3. NumPy
4. Pandas
